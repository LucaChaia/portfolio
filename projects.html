Mancala Engine Project

  • What this is: A Mancala engine implementation created from the framework from Artificial Intelligence: A Modern Approach 3rd Ed. by Peter Norvig and Stuart J. Russell
  • Tech stack: Python, made using Jupyter notebooks and using the random, numpy, dataclasses, time, matplotlib packages.
  • Why I made this: This is a class project that was meant to solidify our understanding of heuristics, decision trees and game theory algorithms.
  • What I learned: We implemented new packages to support data analytics. I learned more about the importance of optimizations, such as when our engine uses the minimax algorithm by itself vs optimizing it by implementing Alpha-Beta pruning to reduce the total number of nodes that need to be searched.
